## Getting Started
```
$git clone https://github.com/silvestreluna/personal-bio-site
```

## Prerequisites
```
$npm install -g http-server
```

## Running 
Browse to personal-bio-site directory and run http server
```
$hs -p 5000
```
Then copy and paste `localhost: 5000` to your browser.
